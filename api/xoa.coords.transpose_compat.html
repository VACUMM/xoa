xoa.coords.transpose_compat¶

xoa.coords.transpose_compat(da, dims)[source]¶

Transpose an array with compatible dimensions

Incompatible dimensions are ignored

Parameters

  • da (xarray.DataArray) – Array to tranpose

  • dims (tuple(str), xarray.DataArray) – Target dimensions or array with dimensions

Returns

Transposed array

Return type

xarray.DataArray

Example

In [1]: a = xr.DataArray(np.ones((2, 3, 4)), dims=('y', 'x', 't'))

In [2]: b = xr.DataArray(np.ones((10, 3, 2)), dims=('m', 'y', 'x'))

In [3]: transpose_compat(a, ('y', 'x')).dims
Out[3]: ('t', 'y', 'x')

In [4]: transpose_compat(a, b.dims).dims
Out[4]: ('t', 'y', 'x')

In [5]: transpose_compat(a, b).dims  # same as with b.dims
Out[5]: ('t', 'y', 'x')
