xoa.grid.get_edges_1d¶

xoa.grid.get_edges_1d(da, axis=-1, name_suffix='_edges')[source]¶

Get edges of coordinates from centers along a single axis

Parameters

  • da (array) –

  • axis (int, str) – Axis index to work on. May be a dimension name if da is a DataArray

Example

# 1D
In [1]: x = np.arange(3) * 2.

In [2]: print(get_edges_1d(x))
[-1.  1.  3.  5.]

# 2D
In [3]: xx = x[np.newaxis, :, np.newaxis]

In [4]: print(xx.shape)
(1, 3, 1)

In [5]: xxe = get_edges_1d(xx, axis=1)

In [6]: print(xxe.shape)
(1, 4, 1)

In [7]: print(xxe)
[[[-1.]
  [ 1.]
  [ 3.]
  [ 5.]]]

# Xarrays
In [8]: x_ = xr.DataArray(x, dims='x', name='x')

In [9]: print(get_edges_1d(x_))
<xarray.DataArray 'x_edges' (x_edges: 4)>
array([-1.,  1.,  3.,  5.])
Dimensions without coordinates: x_edges

In [10]: xx_ = xr.DataArray(xx, dims=('y', 'x', 't'), name='lon')

In [11]: print(get_edges_1d(xx_, axis='x'))  # with dim name
<xarray.DataArray 'lon_edges' (y: 1, x_edges: 4, t: 1)>
array([[[-1.],
        [ 1.],
        [ 3.],
        [ 5.]]])
Dimensions without coordinates: y, x_edges, t

In [12]: print(get_edges_1d(xx_, axis=1))
<xarray.DataArray 'lon_edges' (y: 1, x_edges: 4, t: 1)>
array([[[-1.],
        [ 1.],
        [ 3.],
        [ 5.]]])
Dimensions without coordinates: y, x_edges, t
